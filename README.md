## Installation

### Installation with rubygems

To install it, you just need to add it to your Gemfile:

```ruby
    gem 'lazy_high_charts'
```

then run

```bash
bundle install
```

to install it.

### legacy notice: if you use rails 2.3.x or 3.0.x or 3.1.x, please use versions less than 1.3.3.
